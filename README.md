# Project Goodbook

A self-hosted social network with 2015 Facebook vibes.  
Username + password only. No email required.

---

## Deploy on InfinityFree (or any PHP + MySQL host)

### What you need
- PHP 7.4+
- MySQL 5.x or MariaDB
- A database already created in your hosting control panel

### Steps

1. **Upload all files** to your public web root (e.g. `htdocs/` or `public_html/`) via FTP or the File Manager.

2. **Visit `setup.php`** in your browser:
   ```
   https://yourdomain.com/setup.php
   ```
   Fill in your database credentials and site URL. Click Install.  
   This creates all tables and writes `config.php` for you.

3. **Delete or rename `setup.php`** after setup is complete — this is important.

4. **Make `uploads/` writable.** On InfinityFree you may need to set the folder permissions to `755` via File Manager → right-click → Permissions.

5. **Visit your site:**
   ```
   https://yourdomain.com/
   ```
   Sign up for an account and go.

## Features

- **Register / Login** — username + password, no email
- **Profile pages** — `/profile.php?u=username`
  - Avatar + cover photo upload
  - Bio + editable intro fields (click icon to cycle emoji)
  - Friends list grid
  - Wall feed
- **Privacy controls** (per user in Settings)
  - Profile: private by default, opt-in public
  - Wall posting: only me / friends / any user / public
- **Friends** — mutual request/accept system with pending notifications on home feed
- **Posts** — text + optional photo, on your own wall or a friend's
- **Likes + Comments** — real-time via AJAX
- **Home feed** — posts from you + friends, newest first
- **People search** — topbar search, live as you type
- **Settings** — display name, bio, privacy, wall permission, password change, account delete

## Notes

- Passwords use PHP `password_hash()` (bcrypt). Safe.
- Images are resized server-side via GD (max 1200px for covers, 400px for avatars, 1200px for post images). If GD is unavailable, the original is stored.
- The `uploads/` folder has an `.htaccess` that blocks PHP execution — don't remove it.
- No email, no password reset by default. If users forget their password, an admin would need to reset it directly in the database:
  ```sql
  UPDATE users SET password_hash = '$2y$...' WHERE username = 'someone';
  ```
  Use `password_hash('newpassword', PASSWORD_DEFAULT)` in PHP to generate the hash.
- InfinityFree free tier has upload limits (~10MB files). Images are resized before storage to stay reasonable.

## Customizing

- **Site name** — set during setup, stored in `config.php` as `SITE_NAME`
- **Colors** — edit the `#3b5998` values in `assets/style.css` to rebrand
- **Cover height** — `#cover-area` height in `assets/style.css`
